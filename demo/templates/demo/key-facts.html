Key facts

This is a section with max 3 columns each with a heading, text and optional icon. The items param expects a list of dictionaries like this:

{
    'icon': [icon url],
    'heading': 'Heading',
    'content': '<p>Some HTML.</p>'
}

An optional section_class can be added to style the container e.g.

{% verbatim %}{% key_facts items=key_facts heading='With three items' section_class='background-stone-30' %}{% endverbatim %}
{% key_facts items=key_facts heading='With three items' section_class='background-stone-30' %} {% key_facts items=key_facts_two heading='With two items' %} {% key_facts items=key_facts_one heading='With one item' section_class='background-stone-30' %} {% key_facts items=key_facts_no_icons heading='With no icons' %}

HTML markup

<section class="great-key-facts padding-top-60">
    <div class="container wrap-tight">
      <h2 class="heading-large margin-bottom-30">[heading]</h2>
      <div class="grid-row">
        <div class="great-key-facts-content column-third">
          <img class="great-key-facts-icon" src="[item.icon]" alt="">
          <div class="great-key-facts-text">
            <h3 class="heading-medium">[item.heading]</h3>
            [item.content (markdown/HTML)]
          </div>
        </div>
      </div>
    </div>
  </section>
