Cookie notice

The cookie notice is displayed at the top of every page. When dismissed it will set the cookie 'hide_cookie_notice'. The JavaScript 'cookie-notice.js' will look for that cookie when the page loads. If it finds it, the cookie notice will not display. This JS requires jQuery to run. When JS is disabled in the browser the cookie notice will always be displayed.
