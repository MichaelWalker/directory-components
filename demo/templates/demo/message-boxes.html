Message boxes

Generic message box

Message boxes can have their text and border colours modified in various ways using utility classes.

{% message_box heading="Complete" description="Your account has been created." heading_level='h3' %} 
{% verbatim %}{% message_box heading="Complete" description="Your account has been created." heading_level='h3' %}{% endverbatim %}

An example using custom classes for box_class and heading_class.

{% message_box heading_class='heading-large grey-text' box_class='border-thin width-two-thirds background-stone-30 border-stone grey-text' heading=".heading-large .grey-text" description=".width-two-thirds .background-stone-30 .border-stone .grey-text" heading_level='h3' %} 
{% verbatim %}{% message_box heading_class='heading-large grey-text' box_class='width-two-thirds background-stone-30 border-stone grey-text' heading=".heading-large .grey-text" description=".width-two-thirds .background-stone-30 .border-stone .grey-text" heading_level='h3' %}{% endverbatim %}

HTML markup

<div class="message-box padding-30 border-medium border-teal background-white">
  <h3 class="box-heading heading-medium teal-text">[heading text]</h3>
  <p class="box-description">[description text]</p>
</div>

Message box with icon

With custom icon.

{% verbatim %}{% message_box_with_icon icon="✕" border_colour="flag-red" heading_class='heading-xlarge flag-red-text' box_class='width-two-thirds background-white flag-red-text' heading=".heading-large .flag-red-text" description=".width-two-thirds .background-white .flag-red-text" heading_level='h3' %}{% endverbatim %}

HTML markup

<div class="message-box-with-icon border-medium border-teal width-full background-white">
  <div class="message-box-icon font-large white-text background-teal" aria-hidden="true">[icon]</div>
  <div class="message-box-content padding-15 padding-30-m">
    <h3 class="box-heading heading-medium teal-text">[heading text]</h3>
    <p class="box-description">[description text]</p>
  </div>
</div>

CTA boxes

CTA boxes work in a similar way but have two columns - one for text and another for a button.

{% cta_box box_id='example-cta-box' heading='Need help?' description='If you need more information please get in touch.' button_text='Contact us' button_url='/' %} 
{% verbatim %}{% cta_box box_id='example-cta-box' heading='Need help?' description='If you need more information please get in touch.' button_text='Contact us' button_url='/' %}{% endverbatim %}

These can be modified with utility classes in the same way as message boxes.

{% cta_box heading_class='heading-large grey-text' box_class='border-stone border-thick background-white grey-text' button_class='button-arrow-large' box_id='example-cta-box-two' heading='.heading-large .grey-text' description='.border-stone .border-thick .background-white .grey-text' button_text='.button-arrow-large' button_url='/' %} 
{% verbatim %}{% cta_box heading_class='heading-large grey-text' box_class='border-stone border-thick background-white grey-text' button_class='button-arrow-large' box_id='example-cta-box-two' heading='.heading-large .grey-text' description='.border-stone .border-thick .background-white .grey-text' button_text='.button-arrow-large' button_url='/' %}{% endverbatim %}

HTML markup

<div id="example-cta-box" class="cta-box padding-45 background-stone-60">
  <div class="grid-row">
    <div class="column-half-m column-full">
        <h3 class="box-heading heading-medium">[heading text]</h3>
      <p class="box-description">[description text]</p>
    </div>
    <div class="column-quarter-xl column-half-m column-full button-container">
      <a id="example-cta-box-button" class="button button-blue" href="[button url]">[button text]</a>
    </div>
  </div>
</div>
