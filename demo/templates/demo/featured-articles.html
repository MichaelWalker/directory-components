Featured articles

This is a list of article previews with text and an image for the first item. The items param expects a list of dictionaries like this:

{
    'title': 'Article title',
    'url': '/article',
    'type_of_article': 'Advice',
    'teaser': 'This is some intro text',
    'image': {
        'url': '/link-to-the-image.jpeg',
    },
    'image_alt': 'Some descriptive image text',
}

Used below:

{% verbatim %}{% featured_articles items=articles_list heading="What's new" %}{% endverbatim %}
{% featured_articles items=articles_list heading="What's new" %}

section_class can be given to change the container style:

{% verbatim %}{% featured_articles items=articles_list heading="What's new" section_class='underline-flag-red background-stone-30' %}{% endverbatim %}
{% featured_articles items=articles_list heading="What's new" section_class='underline-flag-red background-stone-30' %} {% endblock content %}
