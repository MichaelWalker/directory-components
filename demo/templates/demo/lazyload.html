Lazyload

Include a single image element in the body of the tag. Use it so that images will load after the rest of the page loads to increase page load times.

{% verbatim %}{% lazyload %}
  <img src="/foo" />
{% endlazyload %}{% endverbatim %}

For background-image elements, you need to make sure it's the child of a noscript element, and the noscript element is the only child of a parent container. The lazysizes library will move the div out of the noscript tags when the user scrolls near.

{% verbatim %}<div class="lazyload ls-noscript" data-noscript="" aria-hidden="true">
  <noscript>
    <div style="background-image: url('{{ img_url }}')"></div>
  </noscript>
</div>{% endverbatim %}
