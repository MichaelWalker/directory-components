Pagination

Pagination requires your page's view to create it's own Pagination object and pass this into the component. The view should include something like: 
{% verbatim %}objects_being_displayed = []
page_size = 5

@property
def pagination(self):
    paginator = Paginator(self.objects_being_displayed, self.page_size)
    return paginator.page(current_page_num or 1){% endverbatim %}
