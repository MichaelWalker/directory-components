Hero

Load this template tag to use the hero.

{% verbatim %}{% load hero from directory_components %}{% endverbatim %}
{% hero background_image_url=hero_image_domestic hero_text="Large title" large_title=True %} 
{% verbatim %}{% hero background_image_url=hero_image_domestic hero_text="Large title" large_title=True %}{% endverbatim %}

Use the hero with large title for when the title is very short. If CMS content is being used, a character count can be done on the title field to decide whether to pass large_title=True to the template.

{% hero background_image_url=hero_image hero_subheading="Exporting to" hero_text="Germany" large_title=True %}
