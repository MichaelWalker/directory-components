Hero

Load this template tag to use the hero.

{% verbatim %}{% load hero from directory_components %}{% endverbatim %}

Responsive image sizes

4 image sizes can be passed to the hero template tag: xlarge_image, large_image, medium_image and small_image.

Breakpoints and images are as follows:

Screen width Image to use Image dimensions
> 1024px xlarge_image 1200x300
> 768px large_image 1024x300
> 640px medium_image 768x300
<= 640px small_image 640x300

medium_image must be provided and is the default image size.

Subheadings

Hero title can also be split into heading and subheading.

{% hero large_image=hero_large medium_image=hero_medium hero_subheading="Exporting to" hero_text="Germany" %} 
{% verbatim %}{% hero large_image=hero_large medium_image=hero_medium hero_subheading="Exporting to" hero_text="Germany" %}{% endverbatim %}

Description

A description can be added after the hero heading.

{% hero hero_text="Hero banner with a description" large_image=hero_large medium_image=hero_medium description='Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.' %} 
{% verbatim %}{% hero hero_text="Hero banner with a description" description='Lorem ipsum...' %}{% endverbatim %}

No image

If an image is not provided a background colour will be used.

{% hero hero_text="Hero banner with no background image" %} 
{% verbatim %}{% hero hero_text="Hero banner with no background image" %}{% endverbatim %}

HTML markup

<section class="great-hero" id="hero">
    <picture class="great-hero-image">
      <source media="(min-width: 1025px)" srcset="[xlarge_image] 1200w">
      <source media="(min-width: 769px)" srcset="[large_image] 1024w">
      <source media="(min-width: 641px)" srcset="[medium_image] 768w">
      <source media="(max-width: 640px)" srcset="[small_image] 640w">
      <img alt="" src="[medium_image]">
    </picture>
    <div class="container">
      <div class="great-hero-title">
          <h1 class="great-hero-heading" id="hero-heading">[heading]</h1>
      </div>
    </div>
  </section>
